ExileTrade is an advanced Path of Exile item search tool that is based on the concept of typing community jargons terms like 400pdps, 2h and 100life.

An index of these terms or Search Terms can be reviewed here. Below are a few example search:

  1. 5L pureev chest 90life 80res
  2. shavs 4ex
  3. 24leveling
  4. (ring or ammy) 20-30fireres 20manaregen 20str
  5. 350pdps mace -voidhome 10c

Questions? Feel free to contact us on the forums, on the gitter chat and on reddit.

First thing is to set the league you want to search. Click the cog button on the left side to select your default for:

  1. League
  2. Buyout Only
  3. Online Only

I want to search for a ring with at least 70 life, 60 total elemental resistances and 30% increased weapon elemental damage. Well, it's a ring, so we type ring. Life is simply life, so we add 70life, total resistances is usually refered as totaleleres or eleres, so we can type any of those: 60eleres. Increased Elemental Damage with Weapons is refered to as wed, so we add wed to our search. That sums up to ring 70life 60eleres 30wed.

Online status is determined as follows:

  1. Player was online in the Ladder API for the past 5 mins. Only the top 15 thousand players are tracked on the ladder while there are about 55k unique accounts with public stash tabs.
  2. Player had any activity with their public stash tab in the last 18 minutes.

If you want to search for a unique item, you simply have to type its name, without spaces.

Example: If I want a Kaom's Heart, i'll have to type kaomsheart, kaomheart or kaom'sheart. As you guys can see, there is space for variations on it. Apostrophes and stuff like the and of are optional.

If you want to search for sockets and links, it's also pretty easy.

Example: I want a 5 linked Lightning Coil. I just have to type: lightningcoil 5l. But honestly I want it to already come with the colors I need, 2 red, 2 greens and 1 blue. So I type: lightningcoil 5l 2r 2g 1b. Why not have all sockets on it? lightningcoil 6s 5l 2r 2g 1b.

Searching for gems works in the same way as searching for uniques, type the name of the gem without spaces.

Example: I want a Blade Vortex: bladevortex. I want it to be at least level 18: bladevortex glvl18. I also want its quality to be 15+: bladevortex glvl18 q15

You can also add b/o's to your search, if you only want to see items under a determined price.

Example: I want a 6l Vaal Regalia for 1 exalt. vaalregalia 6l 1ex

If you don't want an item to have a specific mod, life life because you're CI, you just need to type -life and the results will only display items without life. You can negate any existing search term, so if you want a nice mace but doesn't want to see Voidhomes, you just have to type -voidhome

Exile Tools makes a lot of apparently impossible searches possible. If you need gloves OR boots with 80eleres and 90life, (gloves or boots) 90life 80eleres. Notice that you need to put the items inside parens. You can even do advanced searches like (25fireres 25coldres 25lightres 80res) OR (50res 20int) -unique

To do an automated search, you just need to type your search parameters and click on the Auto icon on the right side of the search bar. It will save your search to the saved searches tab, that can be accessed by clicking the Flag icon, also on the right side of the search bar.

Right now, after saving it, it will just run every entry on your "Automated Search List" for every N secs (30 sec atm). Every entry will then get a count (see screenshot), there's 1 tabula that's priced 10c. If the total count of all entries is > 0, a sound will be played.

Note that automated search doesn't do "online only search" - this will be configurable when we change the "online only" as a search term. Also unlike Durian, it doesn't yet know if any items are new relative to previous run - but this can be worked around using negation; for example: we don't wanna see items from account or ign TestOnlyAccountName, you just add negation to seller search term like this: pureev chest 80res 70life 1300ev -sellerTestOnlyAccountName

Note that the automated search is still in experimental stage.
